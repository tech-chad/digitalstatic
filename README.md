### Commands
- Q = to quit
- b = black and white mode
- C = color mode
- c = cycle color mode
- d = reset to default
- 0 to 9 = Delay

#### Colors
- r = red
- t = green
- y = blue
- u = yellow
- i = magenta
- o = cyan
- p = white
- [ = black
